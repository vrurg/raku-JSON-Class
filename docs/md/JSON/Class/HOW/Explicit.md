# NAME

`JSON::Class::HOW::Explicit`

# METHODS

  - **`method json-is-explicit(Mu)`**
    
    Tell if a class or a role is explicit.

  - **`method json-has-explicit(Mu)`**
    
    *False* if the flag was not explicitly set for the class.
